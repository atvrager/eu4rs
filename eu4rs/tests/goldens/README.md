# Golden Snapshots

This directory contains reference images (goldens) for UI regression testing. These are generated by `ui::tests`.

## Files

### `ui_render_complex.png`
**Test:** `test_render_snapshot`
**Purpose:** Verifies the standard UI layout during gameplay.
**Expected Appearance:**
- **Sidebar (Right):** Dark panel showing province details ("Stockholm").
- **Tooltip (Bottom-Left):** Floating box with hover text ("Hovering...").
- **Map Mode (Top-Left):** Indicator showing "Political".
- **Background:** Transparent (alpha 0) where the map would be.

### `ui_render_full.png`
**Test:** `test_render_full_ui`
**Purpose:** Verifies the UI with **ALL** overlays enabled, specifically the debug console.
**Expected Appearance:**
- **Console Overlay (Top 50%):** A semi-transparent dark overlay covering the top half of the screen.
- **Console Logs:** Text logs ("[INFO] System started", etc.) visible within the console overlay.
- **Sidebar (Right):** Visible, potentially partially obscured or overlaying the console depending on draw order.
- **Background:** Transparent where not covered by UI elements.

### `inspector_province_1.png`
**Test:** `test_province_inspector`
**Purpose:** Verifies the standalone province inspector window.

### `map_*.png`
**Test:** `ops::draw_map_*`
**Purpose:** Reference outputs for the map rendering pipeline (Political, Religion, Trade Goods, etc.).
